Description (EN)

All the projects I did during my Master degree (1st and 2nd year) at Université Pierre et Marie Curie (Paris VI) :

  • Solving Helmholtz equation and wave equation with finite elements P1 (1st year)
  • Solving Navier-Stokes equations with finite elements P1/P2 and method of characteristics (2nd year)
  • Low-rank factorization of dense matrix in parallel (2nd year)

For each project there is a pdf rapport and a readme unfortunately they are written in French. If you have trouble using those programs or questions about it don't hesitate to send me an email. All those projects are written in C or C++.

Description (FR)

Les différents projets que j'ai réalisé en Master (1ère et 2ème année) à l'Université Pierre et Marie Curie (Paris VI) :

  • Résolution de l'équation de Helmholtz et de l'équation des ondes en 2D avec des éléments finis P1 (M1)
  • Résolution des équations de Navier-Stokes avec des éléments finis P1/P2 et la méthode des caractéristiques (M2)
  • Factorisation Low-Rank en parallèle d'une matrice dense (M2)

Pour chaque projet, il y a le rapport au format pdf et un readme pour faire fonctionner le programme. Tous ces programmes sont écrits en C ou C++.
